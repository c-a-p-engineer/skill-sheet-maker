# cline-template

これは、Cline を使用して開発するためのテンプレートリポジトリです。

## 使い方

1.  このテンプレートを使用して新しいリポジトリを作成します。
2.  作成したリポジトリをローカルにクローンします。
3.  Visual Studio Code で開き、Dev Container で再度開くことを推奨されたら従います。
4.  快適な開発をお楽しみください！
